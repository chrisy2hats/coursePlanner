User Guide
About
This website is a tool for planning of university courses.Each course can have as many weeks as you want and every week has topics,notes and resources areas where you can jot down your thoughts.
User Guide
Adding a course
You can add courses by entering the name of a course into the text box called "Course Name" and then pressing the + button.
Adding weeks
To add a week click on the + button to the right of "Weeks". This will add a week to the course that is currently selected
Adding topics
To add a topic click on the + button to the right of "Topics". This will add a area to write your topics for the week.
Adding notes and ideas
To add a area to write down your notes and ideas click the + button to the right of "Notes & Ideas". This will add an area to write notes and ideas for the week.
Adding resources
To add a area to write down any resources for the week such as website you need or links to a slideshow click the + button to the right of "Resources". This will add an area to write the resources for the week.
Transfering ownership of a unit
You can transfer the ownership of a course meaning that you will no longer be able to access the unit however the person you give the unit to will be able to access it including the weeks and the weeks contents.To transfer a course selected it from the courses dropdown, enter the name of the new owner in the text field titled "New Owner Name" and click the button marked "Transfer Ownership".
Changing to dark theme
If you would prefer a black background to the default white one you can click the button titled "Join the dark side!" to change to a black background. You can undo this change by pressing the button again however the text will now say "Join the rebel alliance".
Saving changes
Any changes you make are saved when you change week, change course or when you press the button titled "Save" at the bottom of the screen.
Note to those using screen readers
Effort has been made to make the site friendly by using aria tags where possible.
